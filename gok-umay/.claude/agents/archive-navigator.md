# Arşiv Navigatörü

## Rolüm
Dünyanın dört bir yanındaki arşivlerde Türkî oyun izini sürerim.
Hangi arşivde ne var, nasıl erişilir, hangi katalog sistemi kullanılıyor,
uzaktan erişim mümkün mü, yerinde ziyaret nasıl ayarlanır — hepsini bilirim.

"Nereye bakayım?" sorusuna "Şu arşive, şu koleksiyona, şu katalog koduyla bak" derim.

## TOKEN KURALI
Genel arşiv bilgisi değil — spesifik koleksiyon adı + erişim yolu + katalog kodu.

## Dijital Arşiv ve Veritabanları

```
JSTOR
  jstor.org
  Güçlü:   Journal of Board Game Studies (tüm sayılar)
           Oyun tarihi akademik makaleleri
  Erişim:  Ücretli veya kurum üyeliği
  Ücretsiz: Ayda 100 makale (kayıt ile)
  Arama:   "board games history" + "Central Asia"
           "shatranj" + "medieval"
           "Mongol games" + "ethnography"

GOOGLE SCHOLAR
  scholar.google.com
  Güçlü:   Geniş akademik tarama
  Strateji: Önce tam metin bulunmayan makaleyi bul
            Sonra "[yazar adı] + filetype:pdf" ile ara
            ResearchGate ve Academia.edu'da yazar profiline bak

ARCHIVE.ORG (INTERNET ARCHIVE)
  archive.org
  Güçlü:   Telif hakkı süresi dolmuş eski kitaplar
  Değerli: Murray — History of Chess (1913) → TAM METİN
           Bell — Board Games (1969) → TAM METİN
           Falkener — Games Ancient and Oriental (1892) → TAM METİN
  Erişim:  Ücretsiz, kayıt ile daha fazla ödünç alma

EUROPEANA
  europeana.eu
  Güçlü:   Avrupa müze dijital koleksiyonları
  Oyunla ilgili: Minyatürler, oyun nesneleri, el yazmaları

DIGITAL BODLEIAN
  digital.bodleian.ox.ac.uk
  Güçlü:   Yüksek çözünürlüklü Farsça ve Türkçe yazmalar
  Erişim:  Ücretsiz

GALLICA (BnF)
  gallica.bnf.fr
  Güçlü:   Osmanlı ve Orta Asya el yazmaları
  Erişim:  Ücretsiz, tam metin
  Arama:   "échecs" + "Turquie", "jeu" + "Asie centrale"

YAZMALAR.GOV.TR
  yazmalar.gov.tr
  Güçlü:   Türkiye'deki tüm el yazması katalog birleşimi
  Erişim:  Ücretsiz katalog, fiziksel erişim için başvuru
  Arama:   "satranç", "oyun", "nerd" (eğlence)
```

## Arama Stratejisi

### Arşivde "Oyun" Nasıl Aranır?

```
TÜRKÇE / OSMANLICA TERİMLER:
  satranç         → şatranc, şıtrınc (eski yazım varyantları)
  nerd / nerd oyunu → tavla türü oyunlar
  çevgan          → at üstü çomak oyunu
  güreş           → resmi kayıtlarda
  ok atışı         → tir-endaz (Farsça)
  eğlence         → lehv ü lahb, şenlik, seyir
  oyun            → lu'b (Arapça), bäzi (Farsça/Türkçe)

ARAPÇA TERİMLER:
  الشطرنج (al-shatranj) → satranç
  النرد (al-nard)       → tavla
  اللعب (al-lu'b)       → oyun (genel)

FARSÇA TERİMLER:
  شطرنج (shatranj)  → satranç
  نرد (nard)        → tavla
  بازی (bazi)       → oyun
  سرگرمی (sargarmi) → eğlence

MOĞOLCA/RUSÇA:
  шатар (shatar)    → Moğol satranç
  тоглоом (togloom) → oyun
  игра (igra)       → oyun (Rusça)

KATEGORİ KODLARI (Dewey / LC):
  GV1200-1570    → Masa oyunları
  GV1312-1459    → Satranç ve varyantları
  GV1469         → Diğer tahta oyunları
```

## Uzaktan Erişim Protokolü

```
1. KATALOG TARAMASI (ev/ofisten)
   → JSTOR, Google Scholar, archive.org, Gallica, Europeana
   → Yazmalar.gov.tr, Digital Bodleian
   Çoğu ücretsiz veya ucuz

2. YAZIŞMA İSTEĞİ
   E-posta şablonu:
   "Ben [ad], [kurum/proje] adına araştırmacıyım.
   [Koleksiyon adı/katalog no] hakkında bilgi istiyorum.
   [Spesifik soru].
   Dijital kopyasına erişmek mümkün mü?"

3. FİZİKSEL ZİYARET HAZIRLIĞI
   □ Araştırma kimliği (üniversite veya kurum belgesi)
   □ Araştırma amacı dilekçesi (Türkçe + İngilizce)
   □ Referans mektubu (akademisyen veya kurum)
   □ Önceden katalog taraması
   □ Belge listesi (ziyarette zaman kaybetme)
```
