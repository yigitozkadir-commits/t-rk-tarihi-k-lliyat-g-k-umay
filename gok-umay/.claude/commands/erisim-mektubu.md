# /erisim-mektubu

Arşiv veya akademik kurum için erişim mektubu yaz.

## Kullanım
```
/erisim-mektubu [kurum adı]
/erisim-mektubu [kurum] --dil=[TR|EN]
/erisim-mektubu [kurum] --amac=[arsiv|isbirligi|fon]
```

## Adımlar
1. academic-collaboration'ı çağır
2. Kuruma özel mektup üret
3. Gerekli belgeler listele
4. Takip adımları belirt

## Çıktı
→ Hazır mektup metni (TR + EN)
→ Eke eklenecek belgeler listesi
→ Tahmini yanıt süresi
→ Takip stratejisi
