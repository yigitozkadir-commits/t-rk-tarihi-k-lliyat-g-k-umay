# /arsiv-ara

Belirtilen konu için hangi arşivlere bakılmalı, rehber sun.

## Kullanım
```
/arsiv-ara [oyun adı veya konu]
/arsiv-ara [konu] --ulke=[TR|GB|RU|UZ|MN]
/arsiv-ara [konu] --dijital  (sadece online erişimliler)
```

## Adımlar
1. archive-navigator'ı çağır
2. Konuya göre arşiv haritası çıkar
3. Dijital erişimlileri önce listele
4. Fiziksel ziyaret gerekenleri belirt

## Çıktı
→ Öncelikli arşiv listesi
→ Katalog kodları ve erişim bilgisi
→ Arama terimleri (TR/AR/FA/RU/MN)
