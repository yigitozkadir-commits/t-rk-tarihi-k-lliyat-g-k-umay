# /haftalik-ozet

Bu haftanın AR-GE faaliyetlerini özetle.

## Kullanım
```
/haftalik-ozet
/haftalik-ozet --format=[tam|sunum]
```

## Adımlar
1. Bu haftaki belge analizlerini derle
2. Alınan kararları listele
3. arge-reporter ile Haftalık Özet üret

## Çıktı
→ Haftalık AR-GE Özeti (.md)
→ Gelecek hafta görev listesi
