# /kaynak-dogrula

İki kaynağı karşılaştır, hangisi güvenilir karar ver.

## Kullanım
```
/kaynak-dogrula [kaynak A] vs [kaynak B]
/kaynak-dogrula [şüpheli bilgi]
```

## Adımlar
1. Her iki kaynağı tanı
2. source-validator'ı çağır
3. Karar + gerekçe üret

## Çıktı
→ Kaynak Doğrulama Raporu
→ KABUL / ŞARTLI / ASKIYA AL / RED kararı
→ AR-GE etkisi
