# /kaynak-indir

archive.org'dan oyun tarihi kaynağı indir.

## Kullanım
```
/kaynak-indir murray        (History of Chess)
/kaynak-indir bell          (Board Games)
/kaynak-indir falkener      (Games Ancient and Oriental)
/kaynak-indir [archive.org ID]
```

## Adımlar
1. corpus-builder → internetarchive ile indir
2. Metin versiyonu varsa önce al, yoksa PDF
3. citation-manager → Zotero'ya ekle
4. Kategori A'ya gönder (belge-oku için hazır)
