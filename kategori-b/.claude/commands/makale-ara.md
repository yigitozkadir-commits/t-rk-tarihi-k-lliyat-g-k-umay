# /makale-ara

20+ akademik kaynakta oyun tarihi makalesi ara.

## Kullanım
```
/makale-ara [konu]
/makale-ara [konu] --kaynak=[jstor|semantic|openalex]
/makale-ara [konu] --acik-erisim  (sadece ücretsiz PDF)
/makale-ara [doi]  (DOI çözümleme)
```

## Adımlar
1. research-analyst → paper-search-mcp + Academix
2. Açık erişim PDF varsa önce listele
3. Bulunanları citation-manager → Zotero'ya ekle

## Çıktı
→ Makale listesi (başlık, yazar, yıl, atıf sayısı)
→ PDF linkleri (açık erişim)
→ Zotero'ya eklenenler
