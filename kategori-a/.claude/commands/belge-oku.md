# /belge-oku

Yüklenen belgeyi document-archaeologist ile analiz et.

## Kullanım
```
/belge-oku [dosya veya URL]
/belge-oku [dosya] --oyun=[oyun adı]
/belge-oku [dosya] --hizli  (sadece oyun içeriğine bak)
```

## Adımlar
1. Belge türünü tanı (PDF / görüntü / web / metin)
2. document-archaeologist'i çağır
3. Keşif Raporu üret
4. arge-director'a ilet (otomatik)

## Çıktı
→ Belge Arkeoloji Raporu
→ Otomatik AR-GE öncelik güncellemesi
