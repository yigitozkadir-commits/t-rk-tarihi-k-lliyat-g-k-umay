# /yazi-tani

Yüklenen görüntüdeki yazı sistemini tanı.

## Kullanım
```
/yazi-tani [görüntü]
/yazi-tani [görüntü] --hizli  (sadece sistem adı)
```

## Adımlar
1. script-decoder'ı çağır
2. Yazı sistemi, dönem, dil tespit et
3. Okuma güçlüklerini listele
4. manuscript-reader'a yönlendir

## Çıktı
→ Yazı sistemi + dönem tahmini
→ Önerilen okuma aracı
→ Sonraki adım
