# Belge Arkeolojisi Skill

## Desteklenen Belge Formatları
- PDF (dijital veya taranmış)
- Düz metin (TXT, MD)
- Web sayfası (URL ile)
- Görüntü (el yazması, minyatür)
- DOCX / akademik format

## Dil Desteği
Türkçe, İngilizce, Arapça, Rusça, Moğolca, Kazakça, Özbekçe
Osmanlıca → önce Transkribus ile ön işleme önerilir

## Etiket Sistemi
```
[BELGELENMİŞ]     → Kaynakta açıkça yazıyor
[YORUM]            → Mantıklı çıkarım, kesin değil
[EKSİK]            → Bilgi yok
[ÇELİŞKİ]          → Başka kaynakla çelişiyor
[REKONSTRÜKSIYON]  → Yeniden inşa edilmiş kural
```

## Güvenilirlik Skalası
⭐⭐⭐⭐⭐ Birincil kaynak — orijinal dönem belgesi
⭐⭐⭐⭐   Akademik — hakemli yayın
⭐⭐⭐     İkincil — derleme, popüler akademik
⭐⭐       Belirsiz — atıf zinciri kırık
⭐         Şüpheli — doğrulanamaz

## Rekonstrüksiyon Kuralları
1. Uydurma yasak — her zaman [REKONSTRÜKSIYON] etiketi
2. Benzer oyunlardan türet + gerekçeni yaz
3. Birden fazla seçenek varsa hepsini sun
4. Son kararı her zaman insana bırak
